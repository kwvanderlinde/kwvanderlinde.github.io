The following table shows a list of entities that are usable inside MapTool’s chat window, dialogs, and frames.

Note that some of the entities on this page are from HTML 4.0 and the library used by MapTool currently supports HTML 3.2, so there may be some that don’t work properly.

To use these in your MapTool HTML, choose either the Name or the Code field. Add an ampersand prefix (&) and a semi-colon suffix (;). For example, &hearts; and &#9829; both generate ♥.
