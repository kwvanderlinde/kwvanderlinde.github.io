MapTool v1.11.2 has been released and can be downloaded from the [MapTool GitHub repo](https://github.com/RPTools/maptool/releases/tag/1.11.2).

This release brings a number of new features:
- New VBL modes Pit and Hill.
- New MapTool Easy Connect server option to simplify player connections using Public Key authentication.
- New Add-On library support supporting embedded HTML, CSS, JavaScript, assets, JSON data, MTScript macros and more.
- New server options to disable various GUI elements on the player clients.
- New macros for creating tokens.
- And more. See the [Change Log](https://github.com/RPTools/maptool/blob/1.11.2/CHANGE_LOG.md) for more details.
